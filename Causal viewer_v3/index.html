Guide

How The Viewer Works

Quick Start

Import a graph first. Once a graph is loaded, the viewer becomes a small workbench for filtering, styling, layout adjustment, legend handling, session save/load, and export.

Import

Choose the source format, select a file, and click Import Graph. The browser sends the raw file to the local Python import service, which normalizes it into the viewer's internal graph format.

Targets And Scope

Use the target list to focus analysis on selected nodes. The scope control can keep the full graph visible or reduce the view to neighbors, parents, children, ancestors, or descendants.

Edge Filters

The threshold removes weaker edges. Transparency, width, and label controls can expose weight or confidence values without changing the underlying graph.

Layout And Editing

Drag nodes to reposition them. Hold Edge Pull or Vertex Push to nudge the layout. Right-click a node or edge to edit visual styling directly inside the viewer.

Legend, Sessions, And Export

Use Import Legend to load the matching sample legend. Save sessions to preserve graph state and UI state. Export figures as PNG, JPEG, WebP, or SVG, or export the current graph layout as JSON.

Canvas Controls

Use the mouse wheel to zoom, drag empty space to pan, and drag the divider to resize the sidebar. The viewer remembers key interface settings in the browser.
